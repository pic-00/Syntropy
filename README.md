# 🌌 Conway's Game of Life — Live Wallpaper for Cinnamon

Красивые интерактивные живые обои для рабочего стола Linux Mint (Cinnamon), моделирующие легендарный клеточный автомат «Жизнь» Джона Конвея.

![Основной скриншот](https://github.com СНИМОК)

---

## ✨ Особенности / Features

* **⚡ Высокая производительность:** Минимальное потребление ресурсов процессора и оперативной памяти.
* **🎨 Кастомизация:** Поддержка изменения цветов клеток, сетки и фона.
* **🔄 Авто-рестарт:** Система автоматически перезапускает симуляцию, если популяция вырождается или стабилизируется.
* **🖥️ Полная интеграция:** Идеально работает в окружении Cinnamon, не перекрывая ярлыки рабочего стола.

## 🚀 Быстрый старт / Installation

### 1. Требования (Зависимости)
Убедитесь, что в вашей системе установлены необходимые пакеты:
```bash
sudo apt update
sudo apt install git python3-pip
```

### 2. Клонирование и запуск
Выполните следующие команды в терминале:
```bash
# Клонируйте репозиторий
git clone https://github.com

# Перейдите в папку проекта
cd Syntropy

# Сделайте скрипт исполняемым
chmod +x install.sh

# Запустите установку
./install.sh
```

## 🛠️ Настройка / Configuration

Вы можете изменить правила игры или внешний вид обоев через конфигурационный файл `config.json`:

```json
{
  "fps": 30,
  "cell_color": "#00FF00",
  "bg_color": "#111111",
  "grid": false
}
```
